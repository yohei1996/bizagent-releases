React

Facebook（Meta）が開発したUIライブラリ。コンポーネントベースでWebアプリを構築。

概要​

  • 公式サイト: https://react.dev/
  • 開発元: Meta（旧Facebook）
  • 用途: Webアプリケーション、UI構築

なぜ必要か​

  • Remotionの基盤技術
  • モダンなWebアプリの標準
  • コンポーネントの再利用で効率的な開発
  • 豊富なエコシステム

基本概念​

コンポーネント​

UIを小さな部品に分割して管理。

function Button({ label }) {
  return <button>{label}</button>;
}

JSX​

HTMLライクな構文でUIを記述。

const element = <h1>Hello, world!</h1>;

State​

コンポーネント内のデータ管理。

const [count, setCount] = useState(0);

BizAgentでの活用​

Remotionで動画を作成する際、Reactの知識があると：

  • カスタマイズがしやすい
  • エラーの原因がわかりやすい
  • より複雑なアニメーションが作れる
