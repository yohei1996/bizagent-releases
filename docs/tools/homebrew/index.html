Homebrew

macOS用のパッケージマネージャー。コマンド一発でツールをインストールできます。

概要​

  • 公式サイト: https://brew.sh/
  • 対応OS: macOS, Linux
  • 用途: 開発ツールやアプリケーションのインストール・管理

なぜ必要か​

macOSには標準でパッケージマネージャーがありません。Homebrewを使うことで：

  • コマンド1行でツールをインストール
  • バージョン管理が簡単
  • 依存関係を自動解決
  • アップデートも一括で可能

インストール​

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

基本コマンド​

# パッケージを検索
brew search パッケージ名

# パッケージをインストール
brew install パッケージ名

# GUIアプリをインストール
brew install --cask アプリ名

# インストール済み一覧
brew list

# アップデート
brew update && brew upgrade

# 不要なファイルを削除
brew cleanup

BizAgentでの活用​

BizAgentの環境構築に必要なツールをインストールする際に使用：

# Node.js
brew install node

# Google Cloud CLI
brew install --cask google-cloud-sdk

# Git
brew install git
